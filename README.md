#Library Management System

A console-based Library Management System built using Core Java.

##Concepts Used
- OOP
- Strings
- StringBuilder
- Generics
- Bounded Generics
- Wildcards
- Iterable
- Iterator
- Collection Framework
- ArrayList
- LinkedList
- Stack

##Features
- Add Books
- Remove Books
- Generate Reports
- Undo Operations
- Custom Iterable Collection

##Tech Stack
- Java 17+
- IntelliJ IDEA
